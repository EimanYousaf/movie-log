# Book Log

This project is a simple book tracking website that lets users keep track of books they want to read, are currently reading, or have already finished.

The goal of the project was to practice using HTML, CSS, and JavaScript to build an interactive website. The website allows users to add books, change their reading status, rate books, and organize their reading list.

## What the website does

Users can:

- add a book with a title, genre, author, year, and reading status
- change the reading status of a book
- delete books from the list
- rate books after finishing them using stars
- search for books by title or author
- filter books by reading status
- sort books by title, year, rating, or when they were added
- see a small confetti animation when a book is marked as completed

The book list is also saved in the browser using localStorage, so the books stay there even if the page is refreshed.

## How it was built

This project was built using:

- HTML for the structure of the website  
- CSS for the design and layout  
- JavaScript for the interactive features

JavaScript is used to add books to the list, update the reading status, sort and search books, and store the books in the browser.

## Files in this project

- index.html – the main page of the website  
- style.css – all of the styling for the page  
- script.js – the JavaScript that handles the functionality
